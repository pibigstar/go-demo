# Go语言内部库使用

- file：操作文件
- flag：读取运行时参数
- goroutine：Go中goroutine各种骚操作
- http：http相关操作
- json：json相关操作
- mail: Go发送邮件
- mysql：Go操作MySQL数据库
- net：网络相关操作（写一个简单的聊天室）
- reflect: Go反射相关操作
- regexp: Go正则表达式操作
- shell: Go执行shell命令
- sort: Go切片排序
- string: Go字符串操作
- sync: Go锁操作
- time: 时间操作
- xml: xml操作
- zip: 压缩解压文件
